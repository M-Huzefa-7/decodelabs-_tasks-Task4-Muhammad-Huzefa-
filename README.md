# Image Recognition Using AI

## Overview
This project uses a pretrained deep learning model (MobileNet) to perform image classification. It predicts the content of an input image.

## Features
- Uses pretrained MobileNet model
- Accepts image input
- Performs image classification
- Outputs predicted label

## How It Works
The image is preprocessed and passed into a pretrained neural network. The model then predicts the most likely class label for the image.

## Technologies Used
- Python
- TensorFlow / Keras
- MobileNet (Pretrained Model)

## Learning Outcome
- Understanding pretrained models
- Image preprocessing
- Deep learning inference
